# Project Statement

## Problem Statement

Many people struggle to track daily expenses and often overspend without knowing where their money goes. Existing solutions like spreadsheets or mobile apps can be complex or require internet connectivity. This project provides a simple, offline console-based tool to quickly log expenses and monitor spending against a monthly budget.

## Scope of the Project

The expense tracker allows users to:
- Log expenses with names, amounts, and categories
- Set and track monthly budget limits
- View categorized spending summaries
- Get daily spending recommendations based on remaining budget

The project focuses on simplicity and ease of use. It does not include data persistence, multi-user support, or advanced analytics.

## Target Users

- College students managing pocket money
- Young professionals tracking personal finances
- Individuals preferring simple, offline tools
- Python learners seeking practical projects
- Anyone wanting quick expense tracking without apps
